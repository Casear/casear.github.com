redis on ubuntu

這邊記錄安裝Redis的過程

在ubuntu安裝Reids蠻簡單的

但裝完後，我還想把它改成用upstart的方式啓動

先把原來的啓動模式取消，停掉預設的init.d script

/etc/redis/redis.conf

建立upstart的啓動script /etc/init/redis-server.conf

再來就可以用upstart啟動了
